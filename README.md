# Wireshark DNS Traffic Analysis
## Overview
This project demonstrates basic network traffic analysis using Wireshark. The focus is on capturing live traffic and filtering DNS packets to observe domain name resolution.
## Tools Used
Wireshark
Kali Linux
## Process
### 1. Raw Traffic Capture
This shows the initial unfiltered network packets captured during normal browsing activity.
### 2. Applying DNS Filter
The filter `dns` was applied to isolate DNS-related traffic.
### 3. Filtered DNS Results
This view shows DNS queries and responses where domain names are resolved into IP addresses.

## Key Learning
Understanding how DNS works in real network traffic
Using Wireshark filters to isolate specific protocols
Observing request and response behaviour in packet analysis
